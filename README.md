# ECD Node.js Test

Este repositorio contiene un script en Node.js para procesar archivos XML.

## Requisitos

- Node.js instalado en tu sistema.

## Instalación

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone https://github.com/DiegoDefault04/ECD_NodeJS_Test.git
2. Navegar al directorio
  cd ECD_NodeJS_Test
4. Instalar dependencias
  npm install
5. Ejecutar el script (Verificar antes que los archivos XML se encuentren en la carpeta XML_files
   node xml_read.js
